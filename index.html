Get Started

Install on Android

Four steps. No account needed beyond your SRM credentials.

✅

About the "Unknown Source" warning — it's completely normal

Android shows this alert for any app installed outside the Play Store — including apps from Google and Samsung. It does not indicate risk. Track-It contains no malware, no spyware, and no hidden trackers. Our Play Store listing is coming soon.

1

Download the APK

Tap Download APK at the top, or visit the GitHub Releases page and download the latest .apk file to your device.

2

Allow Install from This Source

When Android prompts you, tap Settings and enable "Allow from this source" for your browser app.

📍 Settings → Apps → Special App Access → Install Unknown Apps → Chrome → Allow
3

Install

Go back and tap Install. Completes in a few seconds. No sign-up required.

4

Log In with Your SRM Credentials

Enter your SRM portal ID and password. They are sent directly to SRM — Track-It never reads or stores them at any point.

5

Future Updates — Handled In-App

Once installed, you never need to revisit GitHub for updates. The app will notify you when a new version is available and let you update with a single tap, right from inside Track-It.

Privacy & Security

Zero-Credential Architecture

Your password never touches our servers. By design, not by promise.

🚫

No Password Storage — Ever

Your SRM password is transmitted directly to SRM. Track-It never reads, logs, or caches it at any stage.

🔀

Pass-Through Authentication

Track-It is a transparent relay. Credentials flow from your device → SRM. Nothing sensitive passes through our infrastructure.

🪪

Only User ID Stored

We store only your SRM email, account creation date, and last-login timestamp for security and upcomming feature like custom reminder section.

📡

Real-Time, Not Persisted

Attendance, marks, and timetable data are fetched live each session, processed in memory, then discarded — never saved.
